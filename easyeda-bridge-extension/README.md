# MCP Pro Bridge

MCP Pro Bridge connects EasyEDA Pro to a local Model Context Protocol server.

## Features

- Read schematic components, wires, and nets from the active EasyEDA project.
- Inspect board and design data through approved MCP tools.
- Run a read only live smoke report for quick connection checks.
- Keep EasyEDA project data on the local machine by default.

## Usage

Install the EasyEDA extension package, start the local MCP server, then use the MCP Bridge menu inside EasyEDA Pro to connect. After the bridge is connected, use an MCP client to run read tools and diagnostics.

Write operations, when enabled by the server profile, should be reviewed with explicit confirmation before use.

## Privacy

The bridge is designed for local first operation. Do not publish private design files, credentials, personal contact details, or confidential project data in Marketplace content.
